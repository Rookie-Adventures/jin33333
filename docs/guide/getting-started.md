# 快速开始

## 环境要求

- Node.js 22.12.0+
- npm 10.0.0+

## 安装步骤

1. 克隆项目
```bash
git clone <repository-url>
cd your-project
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

## 项目结构
```
project/
├── src/
│   ├── components/     # 组件目录
│   ├── pages/         # 页面目录
│   ├── services/      # 服务目录
│   ├── store/         # 状态管理
│   └── utils/         # 工具函数
├── public/            # 静态资源
└── docs/             # 文档目录
```
